# CallBotCompare Planning Package

This folder contains the handoff package for building `callbotcompare.com`.

## Files
- `CLAUDE_CODE_BUILD_SPEC.md` - product, business, UX, and implementation requirements
- `prisma_schema_proposal.prisma` - proposed database schema
- `FRONTEND_SITEMAP_AND_BUILD_PLAN.md` - route map, UI modules, and build sequence
- `CLAUDE_CODE_PROMPT.md` - copy/paste starter prompt for Claude Code
- `seed-vendors.json` - starter vendor data for MVP seeding
- `HOMEPAGE_COPY.md` - homepage copy blocks and metadata
- `FIRST_10_PAGE_BRIEFS.md` - briefs for the first commercial pages
- `CLAUDE_CODE_TASK_CHECKLIST.md` - implementation checklist for build execution
- `SEED_SCRIPT_TEMPLATE.ts` - TypeScript seed script scaffold for Prisma
- `PUBLISH_READY_COPY.md` - homepage and priority page copy in near-final form
- `SCORING_RUBRIC.md` - exact vendor scoring system and field guidance
- `FRONTEND_VISUAL_DESIGN_SPEC.md` - visual design system, colors, typography, and anti-generic UI guidance
- `HOMEPAGE_WIREFRAME.md` - homepage section layout and wireframe guidance
- `STYLE_TILE_BRAND_BOARD.md` - brand feel, color roles, type, and surface treatment
- `COMPONENT_UI_SPEC.md` - component-level UI and interaction guidance
- `MOBILE_LAYOUT_SPEC.md` - responsive and mobile behavior guidance
- `MASTER_BRIEF_FOR_CLAUDE_CODE.md` - single-file master build brief
- `CLAUDE_FRONTEND_DESIGN_SYSTEM.md` - consolidated frontend design, typography, color, and visual system for Claude
