# Style Tile / Brand Board

Project: CallBotCompare

## Brand impression
CallBotCompare should feel like:
- a trusted advisor
- a calm software buyer guide
- a practical operations-focused brand

Keywords:
- warm
- clear
- grounded
- competent
- structured
- editorial

## Typography
### Heading font
- Montserrat
- use for H1, H2, H3, key CTA emphasis, and navigation highlights

### Body font
- Inter
- fallback: system sans-serif

### Font weight guidance
- H1: 700
- H2: 700
- H3: 600
- body: 400 to 500
- small labels: 500 to 600

## Core palette
### Primary brand colors
- Deep Slate: `#1F3A5F`
- Soft Teal: `#2F7F79`
- Warm Cream: `#F7F4EE`
- White: `#FFFFFF`
- Charcoal: `#1F2933`
- Muted Gray: `#5B6875`

### Accent colors
- Soft Gold: `#D9A441`
- Pale Sage: `#DCE9E2`
- Light Blue Gray: `#D6DEE6`
- Success Green: `#2E8B57`
- Amber: `#B7791F`

## Suggested role mapping
- page background: Warm Cream
- cards: White
- main text: Charcoal
- headings/nav: Deep Slate
- primary buttons and active links: Soft Teal
- highlight accents and selective badges: Soft Gold
- secondary section tint: Pale Sage
- borders/dividers: Light Blue Gray

## Texture and feel
- clean and lightly tactile
- mostly flat surfaces with subtle depth
- soft shadow only where helpful
- not glossy
- not futuristic

## Border radius
- cards: 16px
- buttons: 10px to 12px
- input fields: 10px to 12px
- pills/badges: 9999px but used sparingly

## Shadows
### Preferred
- subtle shadow: `0 8px 24px rgba(31, 58, 95, 0.06)`
- hover shadow: slightly stronger but still restrained

### Avoid
- giant floating-glass shadows
- strong blue glows
- layered fake depth everywhere

## Button style
### Primary button
- background: Soft Teal
- text: White
- hover: slightly darker teal
- shape: softly rounded rectangle
- weight: medium/semibold

### Secondary button
- background: White or Warm Cream
- border: Deep Slate or Light Blue Gray
- text: Deep Slate

## Card style
- white surface
- subtle border
- restrained shadow
- defined internal spacing
- clear title hierarchy

## Table style
- use clear row separators
- sticky header on desktop
- light tinted header background
- readable spacing
- avoid overdesigned data tables

## Icon style
- simple line icons only
- muted stroke colors
- use sparingly
- do not let icons dominate content

## Imagery direction
Preferred:
- UI previews
- tasteful charts or scoring visuals
- abstract line motifs only if minimal

Avoid:
- robot mascots
- cartoon bots
- cliché AI graphics
- cheesy holographic visuals
- fake lifestyle teams

## Voice translated into design
Because the copy is plainspoken and practical, the design should mirror that.

That means:
- clean interfaces
- informative layouts
- no hype visuals
- enough warmth to feel approachable
- enough structure to feel credible

## Example section styles
### Editorial section
- cream background
- left-aligned heading
- comfortable line length
- muted supporting text

### Tool section
- white background
- structured grid/table
- stronger visual hierarchy

### CTA section
- pale sage or cream tint
- clear action path
- no hard-sell styling

## Brand rule
If a design choice feels like it came from a generic “AI startup” template, reject it.
